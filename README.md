## CSS Architecture

The site now supports light, dark, and system color themes. The color system lives in `css/tokens/colors.css`, with shared spacing and type tokens in `css/tokens/variables.css`.

Source styles live in `css/` and are organized into cascade layers:

```
css/
├── tokens/
│   ├── colors.css       — color custom properties
│   └── variables.css    — spacing, typography, radius, shadow tokens
├── base/
│   ├── reset.css        — minimal box-sizing / margin reset
│   └── elements.css     — bare HTML element styles (body, headings, links)
├── layout/
│   └── primary.css      — overall page structure
├── components/
│   ├── nav.css          — .site-nav navigation bar
│   └── cards.css        — .card-grid and .card components
├── utilities/
│   └── utilities.css    — single-purpose helper classes
└── main.css             — layer declaration + all @import statements
```

Layer order: `tokens → base → layout → components → utilities`

## Build Tool

This project uses **PostCSS** with:
- [`postcss-import`](https://github.com/postcss/postcss-import) — bundles all `@import` statements into a single file
- [`cssnano`](https://cssnano.github.io/cssnano/) — minifies the output

The build reads `css/main.css` and writes a bundled, minified file to `dist/styles.css`.

## Getting Started

```bash
# Install dependencies
npm install

# Build once (outputs dist/styles.css)
npm run build

# Rebuild automatically on every save
npm run watch
```

The `dist/` directory is generated by the build and is excluded from version control via `.gitignore`. CI generates it fresh on every push.

## Typography System

The typography tokens live in `css/tokens/variables.css`.

This project uses a major third modular scale with a base size of `1rem`. Smaller text sizes, such as body text and captions, stay fixed for readability. Larger heading and display sizes use `clamp()` so they scale smoothly between narrow and wide screens.

The site uses semantic type tokens such as:

- `--font-size-body`
- `--font-size-caption`
- `--font-size-heading-md`
- `--font-size-heading-lg`
- `--font-size-display`

The custom web font is Merriweather. It is loaded through Google Fonts with `display=swap`, which allows fallback text to appear first while the custom font loads.
